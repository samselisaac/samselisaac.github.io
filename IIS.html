1.      One the Server Manager � Select Manage

2.      Select Add Roles and Features

3.      Select Next > until getting to the Server Roles tab

4.      Select the box for Web Server (IIS)

5.      Select Add Features

6.      Select Next > until getting to the Confirmation tab

7.      Select Install

8.      Wait for the IIS Web Server to install

9.      Look up the IIS on the search bar

10.  Open the IIS

11.  Use the drop downs on the left side � Select WIN-IE7T0R2L2OF (the server machine)

14.  On the right side under Actions

16.  Right-click the iisstart.htm

17.  Select Notepad

18.  Edit the HTML code

19.  Now the site should be updated
