Starting Up the VM and Installing Windows 11

1.      Select Power on above the VM

2.      Select Next

3.      Select Previous Version of Setup in the bottom left corner

4.      Select Next

5.      Select Install now

6.      Select Windows 11 Pro Education

7.      Select Next

You will not reach an error window saying you cannot install Windows 11 due to system requirements

8.      On your keyboard Press Shift + F10

This will open a command prompt

9.      Enter regedit to access the Registry Editor

10. Follow the path: HKEY_LOCAL_MACHINE/SYSTEM/Setup

11. Select the Setup folder

12. In the whitespace of the folder Right click

13. Hover over New

14. Select Key

15. Name the Key LabConfig (case-sensitive)

16. Within LabConfig whitespace Right click

17. Hover over New

18. Select DWORD (32-bit) Value (you will repeat this three times)

19. Name each of the 3 DWORD Values respectively

a.      BypassTPMCheck

b.     BypassSecureBootCheck

c.      BypassRAMCheck

20. Double-click on each DWORD Value one at a time

21. Enter 1 for the Value data:

22. Select OK

23. Repeat this for all 3 DWORD Values

24. Close out of the Registry Editor

25. Close out of the Command Prompt

26. Hit the back arrow in the top left corner of the error screen

27. On the Select the operating system you want to install page Select Windows Pro Education

28. Select Next

29. Select the box I accept the Microsoft Software License Terms. If an organization is licensing it, I am authorized to bind the organization.

30. Select Next

31. Select Custom: Install Windows only (advanced)

32. Select the drive you want windows installed on

33. Select Next
