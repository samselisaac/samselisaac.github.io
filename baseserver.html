Creating Server VM

1.       Log in to ESX Server

2.       Above Virtual Machines select Create / Register VM

3.       Select Create a new virtual machine so that it is highlighted

4.       Select Next in bottom right corner

 

 

5.       Enter server name that will be registered on VM server

6.       For Guest OS family select drop down menu

a.       Select Windows

7.       For Guest OS version

a.       Select Microsoft Windows Server 2022 (64-bit)

8.       Select DataStore1 so it is highlighted

9.       Select Next in the bottom right corner

10.  Select CPU dropdown menu

a.       Select 2

11.  Select Memory dropdown menu

a.       Enter 16

b.       Select GB

12.  Select Hard disk 1 dropdown menu

a.       Enter 90

b.       Select GB

13.  Select Network Adapter 1 dropdown menu

a.       Select VM Network

b.       Select the checkbox for Connect

14.  Select CD/DVD Drive 1 dropdown menu

a.       Select the checkbox for Connect

b.       Select Datastore ISO file

15.  Navigate through the folders through path ISO FILES\Windows Server

a.       Select en-us_windows_server_2025_updated_sep_2025_x64_dvd_6d1ad20d.iso

16.  Select Select in bottom right corner

17.  Select Next in bottom right corner

 

 

18.  Select Finish in bottom right corner

Starting Up VM/ Installing Server

1.       Select Power on above the server console window to power on the VM

2.       To enter the VM, click on the console window or

a.       Select Console, then select preferred method of viewing

 

 

3.       Select language

4.       Select Next

 

5.       Select Next

6.       Select dot to select Install Windows Server

7.       Select checkbox next to I agree everything will be deleted including files, apps, and settings

8.       Select Next

9.       Select dot next to Use a product key

a.       Enter product key V7HPG-CMNJX-X39G7-QPH7Q

10.  Select Next

 

11.  Select Windows Server 2025 Standard (Desktop Experience) so it is highlighted

12.  Select Next

 

13.  Select Accept

14.  Select disk driver you want the server installed on

15.  Select Next

 

16.  Select Install

Now the server is installing

 

 

17.  Enter your chosen password for your Administrator access account

18.  Select Finish

19.  Enter your password to log in to the server

 

20.  Select Accept

21.  Server is now downloaded and ready to configure
