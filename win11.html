Creating the VM on ESXI Server

1.   Select Create / Register VM above Virtual Machines

2.    Select Create a new virtual machine

3.    Select Next in bottom right

 

4.    Enter Machine name in Name box

5.    Select Guest OS family dropdown box

6.    Select Windows

7.    Select Guest OS version

8.    Select Microsoft Windows 10 (64-bit)

9.    Select Next in bottom right

10. Select DataStore1

11. Select Next in bottom right

12. Find the Network Adapter 1 tab toward the bottom of screen

13. Change the tab Dead bits to VM Network for network access

14. Find the CD/DVD Drive 1 tab below Network Adapter 1

15. Select the Datastore ISO file

 

16. Within the Datastore browser follow the path

17. ISO FILES\Windows 11\en-us_windows_11_business_editions_version_24h2_updated_july_2025_x64_dvd_d32522f0.iso (should be the last DVD drive in the Windows 11 folder

18. Select Select in the bottom right corner

19. Select Finish in the bottom right

 

The VM is now created and ready to install Windows on

20. Select Power on at the top of the VM page

 

Starting Windows 11 Installation

1.    Hit enter on highlighted Boot normally

2.    Select Previous Version of Setup in bottom left corner

3.    Make sure that the top two is on English (United States) and the last on US

4.    Select Next

5.    Select Install now

 


6.    Select Windows 11 Pro Education

7.    Select Next

8.    After getting this error message, on your keyboard press Shift + F10

 

9.    Once in the Command Prompt type regedit to access the Registry Editor

10. Select the HKEY_LOCAL_MACHINE

11. Select the SYSTEM folder

12. Select the Setup folder

13. Right click the Setup folder

14. Select New

15. Then select Key

16. Name this key LabConfig (case matters)

 

17. Within LabConfig right click

18. Select Value

19. Select DWORD (32-bit)

20. Name DWORD BypassTPMCheck

21. On your keyboard hit the Enter key

22. Select Value

23. Select DWORD (32-bit)

24. Name DWORD BypassSecureBootCheck

25. On your keyboard hit the Enter key

26. Select Value

27. Select DWORD (32-bit)

28. Name DWORD BypassRAMCheck

29. On your keyboard hit the Enter key

30. Double click on an individual DWORD value you created

31. Under Value data: change the 0 to a 1

32. Select OK in the bottom right corner

33. Repeat this for the other two DWORD values you created

 

34. In the top right corner of the Registry Editor hit the X to close it

35. In the top right corner of the Command Prompt hit the X to close it

36. Back to the error screen, hit the back arrow in the top left corner

37. Select Windows 11 Pro Education

38. Select Next in the bottom right corner

 

39. Select Next in the bottom right corner

40. Select Custom: Install Windows only (advanced)

41. Select whatever drive you want to install windows, in this case it is Drive 0 Unallocated Space

42. Select Next in the bottom right corner

The Windows installation has now started
